Weird Wellington Project

This project was my first attempt at using 3D modelling and was designed to teach us the basic skills required. This project involved taking a section of Wellington and adding a theme to it (Neo Tokyo).

When: Q1 2017

Where: Victoria University of Wellington

Programs used: Maya2017

Skills: 3D modelling, lighting

About the project

This project was my first ever time using 3D-Modelling. This course was designed to teach us the basic techniques such as blocking, extruding and basic lighting. We weren't allowed to do any texturing or mapping as it was just meant to be basic blocked shapes.

For this project we had to take a photo of somewhere in wellington and then mix it one of the selected themes of which I picked Neo Tokyo. The one thing I didnt want to forget when making it Neo Tokyo was the wellington, this is because welligton buildings have a very detailed ?archatecture? and that without those style buildings it would lose its wellington feel. I tried to make it feel as if the street were still very wellington but the high-risers were the futuristic part.
