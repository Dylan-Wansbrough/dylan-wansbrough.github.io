Animation Project

This was a project about learning simple animation in Maya and applying animation rules/techniques as well as simple physics.

When: Q1 2018

Where: Victoria University of Wellington

Programs used: Maya2018

Skills: Animation principles, physics, constraints, skeletons, weight painting, simple rigging

About the project

Over the course of 4 weeks we had to create a number of short animated sequences that will feature an animated tool manipulating an object. It involved creating 6, 7 second animations joined together in a loop to form a sort of "story". It had to involve one object being passed or manipulated between "tools".

This project involved learning basic physics and the fundamentals of animation. This meant I had to learn about squash and stretch as well as anticipation. These were things that were used to create a more living and realistic feel to the animation. The main issue with this project was time, animation on a basic level isnt to hard but it just requires a lot of time and trying to perfect the movement.
