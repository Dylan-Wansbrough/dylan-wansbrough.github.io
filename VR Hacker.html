Cyberpunk Apartment (VR)

Creating a room to explore in VR using Unity.

When: Q4 2019

Where: Victoria University of Wellington

Programs used: Unity 2018.2.1f1, Maya2018, PhotoShop CC, Substance Painter

Skills: VR design, lighting, UV unwrapping, 3D modelling, Texture Painting

About the project

This was made for my for A University project designed to tell a narritive. I have created it so that you can go play this in VR using a Vive. I have implement the ability to teleport, pick up and throw items, open doors and "swallow" select items. This is to try and create a immersive experince.
