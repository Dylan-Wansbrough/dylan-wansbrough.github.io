JavaScript Project

This was a project about learning how to use JavaScript as well as learning about the compostion and layout of an image.

When: Q4 2017 - Q1 2018

Where: Victoria University of Wellington

Programs used: p5.js

Skills: Programming, JavaScript, composition

About the project

This course was designed to teach us the basics of JavaScript while also teaching us how to use it in ways that incorporate design techniques. This included learning about compositions, colour, form and visual perception. While our knowledge of JavaScript was very little we were tasked with designing compositions using simple shapes and techniques.

We were required to make 9 compositions, one of each category. The first and main categories were Ellipse, Lines and Polygon meaning for each composition we could only use those selected shapes. Each of these main categories had 3 subcategories: Grid, Algorithm and Open. Grid meant everything had to be in some sort of grid like pattern, Algorithm required using formulas such as sin/cos or others to create something and open was free to make anything of your choosing with that shape.
