About the project

Most of the games mechanics have been implemented and the game needs a complete visual and SFX redesign.

Beat Master is a game I have been solo developing for almost 2 years now. with a release date of Q3-Q4 2021 on steam. I have hired composers to create music for the game and have worked with them to make music that fits the game and with more to come. During this project I wanted to learn not only how to make a prototype but how to make a fully functioning game and the theory and decisions that go behind each choice. I used a wide array of tools and tutorials to tech myself game design mechanics and skills.

I wanted to create a game not only that people would play but would hook them in, during play testing I found for example that by having the "Q" to reload feature people would quickly get back into it when they lost keeping them engaged and in the loop.
