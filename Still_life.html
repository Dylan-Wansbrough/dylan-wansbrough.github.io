Virtually Still-Life Project

In this project we had to recreate a scene that we created in real life and use it to make a composition that compliments the objects.

When: Q2 2017

Where: Victoria University of Wellington

Programs used: Maya2017

Skills: 3D modelling, lighting, UV unwrapping, texturing

About the project

In this project we had to take 6 items from real life and convert them into a 3D still life image. For this I used my old computer set up. I used a lot of detail for things that didn't end up being visable due to the way I used lighting in the scene. Looking back now I would have used a lighter ambient lighting so that some of the details of the models would have been able to stand out more.

This was my first time working with texturing, UV unwrapping and normal maps.
